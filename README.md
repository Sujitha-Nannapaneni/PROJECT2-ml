# Decision Tree Performance on Multiple Datasets

## Project Overview
This project applies a single machine learning algorithm, Decision Tree Classifier, on multiple datasets and compares its performance.

## Algorithm Used
* Decision Tree Classifier

## Datasets Used
* Iris Dataset
* Wine Dataset
* Breast Cancer Dataset

## Technologies Used
* Python
* Pandas
* Scikit-learn
* Matplotlib

## Steps Performed
1. Loaded multiple datasets
2. Split each dataset into training and testing sets
3. Applied Decision Tree Classifier
4. Calculated accuracy scores
5. Compared results using tables and graphs

## Results
The performance of the Decision Tree algorithm was analyzed on different datasets to determine which dataset is most suitable.

## Conclusion
This project shows that the same algorithm can produce different accuracy scores on different datasets depending on dataset characteristics and complexity.
